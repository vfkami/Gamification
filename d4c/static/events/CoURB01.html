PALESTRA — Artur Ziviani (Laboratório Nacional de Computação Científica) 08:30 – 10:00
Título: Análise de mobilidade urbana com base em dados de telefonia móvel

Resumo: Esta palestra traz uma visão geral do uso de dados de telefonia móvel para análise de mobilidade urbana com diferentes propósitos e combinação com diferentes aspectos. Em particular, a palestra relata resultados de projetos anteriores considerando mobilidade urbana devido à realização de grandes eventos, modelos de predição de mobilidade e associação de mobilidade a aspectos demográficos e sócio-econômicos.
